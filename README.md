# Notification
If you have already installed rEFInd on a computer, it will need to be removed for installation !                                                                  
In order for the program to work, you need to install .NET FRAMEWORK 6.0.1

# Usage
1. Run Program
2. Select install mode (Computer, USB or Desktop unpack)
3. Configure Options
4. Click "Install rEFInd" Button
5. Done!

# Supported
### Operating Systems
1. Windows 7-11 or HackBGRT
2. Maybe all Linux systems
3. PhoenixOS

### Utilities
1. Grub File Manager
